# Тестовое задание

## Описание
Поле `textarea` с поддержкой упоминаний пользователей через `@`.

## Функционал
- При вводе `@` + текст показывается список пользователей (фильтр по ФИО / username).
- Дропдаун позиционируется рядом с курсором, не просто снизу.
- После выбора пользователя вставляется `@username` в текущую позицию курсора.
- Поддержка нескольких упоминаний в одном тексте.
- Список скрывается при потере фокуса, нажатии `Esc` или клике вне элемента.
